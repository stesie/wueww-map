Datenschutz

Diese Map ist auf Datensparsamkeit ausgelegt. Von dir in dieser angeklickte Elemente werden nicht gesammelt oder anderweitig getrackt. Die Kartendaten (Kachelbilder) werden von OpenStreetMap heruntergeladen, die Fahrplandaten initial von GitHub Pages. Die App wird ebenso über GitHub Pages veröffentlicht.
